- **来源**：腾讯云直播
- **日期**：2026-03-25
- **主题**：企业级 AI Agent（"小龙虾"）的构建与落地实践
- **原始文件**：`raw/02-市场情报/行业动态/会议记录/讯飞转写_2026年03月25日_直播笔记：企业级小龙虾养成指南.txt`

## 核心内容

### 1. 腾讯云智能体开发平台 (Agent Development Platform)

腾讯云发布了面向企业的智能体开发平台，主打"企业级"定位。核心观点：
- 企业级 Agent 不能简单套用消费级大模型应用思路
- 强调"任务驱动"而非"对话驱动"
- 需要提供完整的开发工具链、评估体系和运营观测能力

### 2. "小龙虾"隐喻

用"小龙虾养成"比喻企业级 Agent 的构建：
- 需要合适的"水质"（企业数据环境、权限体系、流程规范）
- 不能直接把 Agent "放生"到生产环境
- 强调渐进式培养、持续迭代和人工介入

### 3. Agent 安全架构三层模型

1. **Host 层（宿主机安全）**：运行环境隔离、资源限制、基线检查
2. **Runtime 层（运行时安全）**：调用审计、行为监控、异常检测、沙箱执行
3. **Network 层（网络安全）**：API 边界控制、流量加密、访问鉴权

### 4. 圆桌讨论要点

- 企业落地 Agent 最大的挑战不是技术，而是流程改造和权责界定
- 需要建立 Agent 的效果评估体系（不只是准确率，还有业务指标）
- 多 Agent 协作的调度与冲突解决仍是早期阶段
- 强调"人在回路"（Human-in-the-loop）在关键决策环节的必要性

## 关键引用

> "把小龙虾直接丢进长江是养不成的，企业级 Agent 需要可控的养殖环境。"
