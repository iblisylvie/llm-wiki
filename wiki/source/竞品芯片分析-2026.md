## 一、NVIDIA GPU 技术要点（来自 docx）

### 1.1 中国特供版芯片：H20 / L20 / L2

由于 A800、H800 被禁售，NVIDIA 针对中国市场推出了阉割版：

| 指标 | H20 vs H100 | L20 vs L40 | L2 vs L4 |
|------|-------------|------------|----------|
| 显存/带宽 | 基本不变，H20 带宽甚至从 3.4TB/s 提升到 4.0TB/s | — | — |
| NVLink | H20 保留 900GB/s（与 H100 一致） | — | — |
| Tensor Core FP16 | 1979 : 148 | 181 : 119.5 | 242 : 96.5 |
| CUDA Core FP32 | 67 : 44 | 90.5 : 59.8 | 30.3 : 48.3 |

**结论**：中国特供版核心策略是**保留显存容量、带宽和互联能力，大幅削减算力**。

### 1.2 数据类型演进

- **V100（Volta）**：Tensor Core 仅支持 FP16。
- **A100（Ampere）**：新增 TF32、BF16 支持。
- **H100（Hopper）**：新增 FP8（E5M2 / E4M3），INT4 不再被 Tensor Core 支持。
- **微软 FP8-LM**：论文显示 FP8 训练大模型可比 BF16 加速 64%。

### 1.3 CUDA Core vs Tensor Core 算力

- **CUDA Core**：一个时钟周期完成 1 次 FFMA。A100 FP32 = 6912 × 1410MHz × 2 ≈ 19.5 TFLOPS。
- **Tensor Core**：一个时钟周期完成一组矩阵乘加。A100 FP16 = 432 × 1410MHz × 256 × 2 ≈ **311 TFLOPS**，远超 CUDA Core。

### 1.4 显存：HBM vs GDDR

- **HBM**：高带宽、高价格，常用于训练卡（H100、A100）。
- **GDDR**：性价比高，常用于推理卡和游戏 GPU（T4、RTX 4090）。
- **H200 升级**：显存从 80GB 升级到 141GB，带宽从 3.5TB/s 提升到 4.8TB/s，算力不变，专为**大模型推理**优化。

### 1.5 GPU 间通信技术栈

| 技术 | 作用 | 关键指标 |
|------|------|----------|
| **GPUDirect v1** | GPU 与 IB 网卡共享锁页内存 | 减少 1 次 CPU 拷贝 |
| **GPUDirect v2 (P2P)** | 同一 PCIe 总线上 GPU 直接互访显存 | 绕过系统内存 |
| **GPUDirect v3 (RDMA)** | 跨节点 GPU 直接网络通信 | 零拷贝远程访问 |
| **GPUDirect Storage** | 存储与 GPU 显存直接交互 | 绕过 CPU 内存 |
| **NVLink** | GPU 点对点高速互联 | v4 单向 25GB/s |
| **NVSwitch** | 多 GPU 全互联交换 | v3 总带宽 3.2TB/s |

实测：8 卡 A100 通过 NVSwitch 全互联，单向带宽约 **270 GB/s**，双向约 **500 GB/s**。

## 二、国产及海外 AI 芯片参数对比（来自 Excel）

### 2.1 整体格局

资料覆盖了 12 家厂商、30+ 款产品，覆盖云端推理、训练、训推一体、边缘等场景。

| **NVIDIA** | H20 / L20 / A100 / H800 / RTX 4090 / RTX 5090 | 训练 / 推理 / 游戏 | H20: 296 TFLOPS (FP16) | 48-141GB | 2024 年中国出货超 60% |

### 2.3 关键观察

1. **国产芯片以推理为主，训练能力仍在追赶**
   - 大部分国产芯片（昇腾 910 系列除外）的 FP32/FP16 算力与 A100/H100 仍有显著差距。
   - 寒武纪 MLU590、昆仑芯 P800、天数智芯智铠 100 等开始向"训推一体"延伸。

2. **显存容量和带宽是国产芯片的主要瓶颈**
   - 大模型推理对显存容量敏感，13B 以上模型对单卡显存要求通常在 24GB 以上。
   - 华为 310 系列被反馈"13B 以上模型放不进去"，视觉大模型场景仍首选 NVIDIA。

3. **软件生态是国产芯片的核心短板**
   - 寒武纪：架构和指令集特殊，开源大模型适配工作量大。
   - 登临：SDK 版本间模型不兼容，转换工具错误信息不足。
   - 昇腾 910：FP32 支持不足，必须依赖华为自身软件栈和深度优化。
   - 海光：主打 CUDA 兼容，生态迁移成本相对较低。

4. **价格与供货**
   - 昇腾 910B 八卡服务器价格约 130-150 万元；910C 约 18 万元/卡。
   - NVIDIA H20 八卡服务器约 110-130 万元，与中国国产高端卡价格接近。
   - 昇腾 2024 年 IDC 出货 64W，占据国产最大份额；NVIDIA 仍占中国市场超 60%。
