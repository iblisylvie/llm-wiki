# 概览

本 Wiki 已完成第一批资料摄入，当前聚焦于两个主题方向：**企业级 AI Agent** 与 **Physical AI（物理人工智能）**。

## 当前主要结论

### 1. 企业级 AI Agent 强调"任务驱动"和可控部署

来自腾讯云 2026-03-25 直播的核心观点：
- 企业级 Agent 与消费级 Agent 的最大差异在于**任务驱动 vs. 对话驱动**
- Agent 不能简单"放生"到生产环境，需要渐进式培养（"小龙虾养成"隐喻）
- 安全架构需要覆盖 **Host 层、Runtime 层、Network 层** 三个层面
- 多 Agent 协作、效果评估、人在回路（Human-in-the-loop）是企业落地的关键课题

### 2. Physical AI 正处于"算法-芯片-系统-物理世界"融合的关键期

来自 2026-03-28 清华大学电子工程系系友论坛的核心观点：
- **世界模型**被定位为具身智能的"认知引擎"，但当前在物理一致性、因果推理、长程预测方面仍有显著差距
- **数据稀缺**和 **Sim-to-Real 鸿沟**是具身智能落地的最大瓶颈
- 短期内，**专用机器人**（如康复医疗、工业制造）比**通用人形机器人**更易落地
- 人形机器人被视为"终极形态"，但商业化时间表存在较大分歧（3-10 年以上）

### 3. AI 芯片竞争焦点正从"训练算力"转向"推理效率"和"系统集成"

来自研讨会下午场（芯片与架构方向）的核心观点：
- **推理阶段的核心瓶颈是内存带宽，而非算力**
- 下一代架构的关键技术包括：**存算一体（PIM）、近存计算、3D 混合键合、软件定义芯片、先进封装（2.5D/3D/TSV）**
- 中国产业链在先进封装和 EDA 工具链方面正在加速布局自主可控能力

### 4. 国产 AI 芯片在"量"上已初具规模，但在"质"和"生态"上仍与 NVIDIA 存在显著差距

来自 2026-04-14 竞品资料分析的核心观点：
- **华为昇腾**以 64W 张（2024 IDC）出货量领跑国产，但 910C 价格已看齐 NVIDIA H20，且存在供货紧张和软件栈依赖问题
- **国产芯片的通用性和软件生态是最大短板**：寒武纪指令兼容性差、登临 SDK 不成熟、昇腾 310 难以支持 13B 以上大模型
- **NVIDIA 仍占中国市场 60% 以上**，其优势不仅在于硬件算力，更在于 CUDA 生态、Tensor Core、NVLink/NVSwitch 的完整技术栈
- **海光**凭借 CUDA 兼容性成为生态迁移成本最低的国产替代方案，**AMD MI300X** 则以 192GB HBM3 成为 NVIDIA 在高端市场的有力挑战者

## 值得关注的问题

1. 企业级 Agent 的评估体系和多 Agent 协作协议何时能形成行业共识？
2. 世界模型在物理一致性上的突破还需要哪些技术路径？
3. 国产先进封装和 EDA 工具链能否在 2-3 年内支撑起大规模 AI 芯片量产？
4. 推理优化技术（如 P/D 分离、KV Cache 优化）的产业化落地进度如何？
5. 国产 AI 芯片能否在 2-3 年内补齐软件生态和工具链的短板？
6. NVIDIA 的 H20/L20 等阉割版产品是否足以维持其在中国市场的垄断地位？
7. 海光的 CUDA 兼容路线能否成为国产替代的主流选择？
